The Rotating Clock is a fun widget that spins relative to the local time! The Clock adds a lot of character and individuality to your website.

== Description ==

The Rotating Clock is an animated widget created by [Ferritech](http://www.ferritech.com/designs/wordpress) that spins relative to the local time. The time is displayed at the top of the clock while a disk spins in real-time showing the relative position of the sun for your time-zone.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `Rotating Clock Widget` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the "Rotating Clock Widget" option in the Appearance->Widgets area and install in a widget area.
4. Enjoy!

== Frequently Asked Questions ==

= Can I change the color of the clock font? =

Currently, this is not an option. If you are savvy with HTML/CSS, then you can easily go into the plugin directory and update the '#bg_time' color in the css file to the color that you want.
